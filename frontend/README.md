# Fantasy Football Analysis Frontend

React-based frontend for the Fantasy Football Analysis tool.

## Setup

```bash
npm install
npm run dev
```

The frontend will run on `http://localhost:3000` and proxy API requests to `http://localhost:8000`.

## Features

- Player rankings with format/position/model filtering
- Player search
- Team schedules
- Streaming recommendations
- Matchup analysis

## Build for Production

```bash
npm run build
npm run preview
```
